Squirrelplate

A HTML5 + CSS3 starting point—designed for squirrels and other small clawed creatures.

Responsive websites, grid systems, media queries, HTML5 tags and cross browser/platform testing — it's all very complicated and time consuming, especially if you are a busy little squirrel. Squirrelplate is a simple template allowing you to build a very quick HTML page.

  • HTML5 reset and basic styling
  • Simple Print Styling
  • 100% Fluid — no media queries
